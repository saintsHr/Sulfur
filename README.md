# Sulfur

Sulfur is a systems programming language designed for developers who want **maximum performance and low-level control** without sacrificing **ergonomics, safety, and simplicity**.

Its goal is to stay predictable and lightweight while providing modern language features that reduce boilerplate and help prevent common programming mistakes.

## Goals

- 🚀 Zero-cost abstractions
- ⚡ High performance
- 🎯 Explicit control over behavior
- 🛡️ Safe by default whenever possible
- 🧩 Simple and consistent syntax
- 📦 Minimal language, powerful foundations

## Current Status

Sulfur is currently under active development and is **not yet production-ready**. The language, compiler, and standard library are evolving rapidly, and breaking changes are expected.

## Documentation

The official documentation can be found in the `docs/` directory.

## Example

```sulfur
i32 x = 93;
u64 y;

y = 74;

i16 z = ((x + y) * y) as i16;
```
