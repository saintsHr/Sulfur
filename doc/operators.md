# Binary Operators

Binary operators operate on two operands.

## Arithmetic

| Operator | Description    |
|----------|----------------|
| `+`      | Addition       |
| `-`      | Subtraction    |
| `*`      | Multiplication |
| `/`      | Division       |

### Example

```
i32 a = 10;
i32 b = 3;

i32 sum = a + b;
i32 difference = a - b;
i32 product = a * b;
i32 quotient = a / b;
```

# Unary Operators

Unary operators operate on a single operand.

| Operator | Description         |
|----------|---------------------|
| `-`      | Arithmetic negation |

### Example

```
i32 x_normal = 10;
i32 x_negative = -x;
```
